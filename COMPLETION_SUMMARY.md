# Completion Summary

This repository contains the static HTML/JS/CSS version of the Deaf Sports History web app with Supabase integration and Netlify deployment configuration.

## Latest Progress (v4.2)

- Implemented Core vs Advanced stats UX with `+ Advanced Stats` toggle for basketball, baseball/softball, and football.
- Added football **Receiving** tab and reordered football tabs to: Passing, Rushing, Receiving, Defense.
- Applied sport-specific default core columns to reduce table clutter while preserving all existing advanced metrics.
- Improved filter/tabs layout robustness for zoom changes and long school names to prevent overlap.
- Documented pagination anti-regression policy and modular service-layer fetch strategy for future AI/dev handoff.
