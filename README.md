# National Prep Deaf Sports

Welcome to the National Prep Deaf Sports website with AI Chat Assistant!

## Features

- **AI Chat Interface**: Interactive chat assistant to help users learn about National Prep Deaf Sports
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **User-Friendly**: Easy-to-use chat interface with smooth animations

## How to Use

1. Open `index.html` in your web browser
2. Type your message in the chat input box
3. Click "Send" or press Enter to send your message
4. The AI assistant will respond with helpful information

## Chat Capabilities

The AI chat assistant can help with:
- General information about National Prep Deaf Sports
- Sports programs and activities
- Events and tournaments
- How to join programs
- Contact information

## Getting Started

Simply open the `index.html` file in any modern web browser to start chatting!

## Files

- `index.html` - Main chat interface page
- `styles.css` - Styling for the chat interface
- `chat.js` - JavaScript functionality for the AI chat

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## About

National Prep Deaf Sports is dedicated to providing athletic opportunities for deaf and hard-of-hearing students across the nation.
